HTTP INFO

The foundation of the World Wide Web is the Hypertext Transfer protocol or HTTP for short. It is used mostly for loading pages using Hypertext links. “HTTP is an application layer protocol designed to transfer information between networked devices” (Cloudflare, 2023). The protocol runs on layers of the protocol stack. The flow usually involves a client making a request to the server which then sends a response message.


Understanding the basics of HTTP is intergral to understanding avertising tech. Designed in the 1990's. HTTP is a protocool which evolved over time. It is an application layer sent over TCP or TLS encryoted TCP connection. Due to extinsibility it is used to fetch documents and images.


  • URL: Evryone is Familiar with URL's, written in standard HTTP, provides the means to identify and locate a resource.

  • Scheme: This defines the resource. The scheme tells the server what type of resource the client is looking for and what format the rest of the locator will be in.

  • Port: The Port Number is optional so it is not in most URL's If the port is not listed then the default port for the scheme is used.

  • Path: Is used to define how to find a source. The hostname sends you to the right IP address, and the path further directs how to get to a more specific location.
